指定されたHTMLファイルの検証を実施してください。

対象: $ARGUMENTS

## 検証項目

### 1. HTML構文チェック
- DOCTYPE宣言の有無
- タグの正しい入れ子構造
- 必須属性の有無（img の alt、a の href 等）
- 非推奨タグ・属性の使用

### 2. アクセシビリティ（WCAG 2.1 準拠チェック）
- 見出しの階層構造（h1〜h6の順序）
- フォーム要素の label 関連付け
- 画像の代替テキスト
- 色のコントラスト比への言及
- キーボード操作への配慮

### 3. SEO基本チェック
- title タグの有無と内容
- meta description の有無
- lang 属性の指定
- 見出し構造の適切さ

### 4. パフォーマンス
- 不要なリソースの読み込み
- 画像の最適化への言及

## 出力フォーマット

各項目について Pass / Fail / Warning で判定し、Fail・Warning の場合は改善案を提示してください。
